Aim of project:
- To create a virtual pet with Object-Oriented Programming.
- To use constructors and prototype methods.

Features of the pet will include:
[x] Name of pet.
[x] They can get older.
[x] As they get older, they get hungrier and less fit.
[x] Can increase the fitness of the pet by walking it.
[x] Feed the pet to decrease it's hunger.
[x] Can talk to the pet to see if it needs feeding or walking.
[x] If pet get too hungry or unfit, it will die.
[x] If pet gets to 30days old, it will die.
